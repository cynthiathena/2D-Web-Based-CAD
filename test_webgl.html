Membuat struktur

Gunakan tombol "Create Square" untuk mencetak persegi.
Gunakan tombol "Create Line" untuk mencetak garis.
Gunakan tombol "Create polygon" untuk mencetak poligon / segi banyak.

Gunakan slider untuk mengubah ukuran persegi atau garis.

Poligon

Isi kotak "Enter polygon sides" untuk mengatur jumlah sudut / segi dari poligon.
Isi kotak "Enter polygon length" untuk mengatur ukuran poligon.
Warna poligon dapat dipilih dengan menekan tombol dengan tombol warna

Export/import

Isi nama file pada kotak di sebelah tombol "Safe File" dan tekan tombol "Safe File" untuk menyimpan struktur yang sudah dibuat kedalam file dengan nama yang sudah diisi.
tekan tombol "Choose File" untuk import dari file yang sebelumnya pernah disimpan
